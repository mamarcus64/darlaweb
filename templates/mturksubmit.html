How to record audio on your computer

Mac users

  1. Go to your Applications folder, and open QuickTime Player (or click on the magnifying glass icon on the top right of the screen and search for QuickTimePlayer).
  2. Open the File menu at the top of your screen and select New Audio Recording.
  3. A recorder should appear. Click the red button in the middle to start recording.
  4. Click the square button (where the red button was) to stop recording.
  5. Play back once to make sure that the audio was recorded properly.
  6. Open the File menu at the top of the screen again and select Save…
  7. Type the appropriate file name as indicated in the survey directions in Export As: and choose a location where you will be able to find the file easily in Where:
  8. Upload your recording to the survey from the location you saved it in in the previous step

Windows PC Users

  1. Go to Start, select All Programs, select Accessories, select Entertainment, and finally select Sound Recorder (or click Start and search for Sound Recorder).
  2. A recorder should appear. Click the Start Recording button on the left to start recording.
  3. Click the Stop Recording button (where the Start Recording button was) to stop recording.
  4. A window to save should appear. Navigate to a location where you will be able to find the file easily and type an appropriate file name. Click ‘Save’ and the recording will save in that location.
  5. Open the file and play once to make sure that the audio was recorded properly.
  6. Upload your recording to the survey from the location you saved it in in Step 4.
