DARLA, short for Dartmouth Linguistic Automation, provides a suite of automated vowel formant extraction tools tailored to the research questions of sociophonetics. We invite you to try it on your own data.

Completely Automated Vowel Extraction

Audio → Formants

Assisted by speech recognition

Semi-Automated Alignment and Extraction

Audio + Transcriptions → Formants

Equivalent to the FAVE pipleline, with additional input and output options

System Output

Upload one or more raw audio files of English speech, with or without manual transcriptions. DARLA returns

  1. aligned TextGrids
  2. formant plots representing the speakers' vowel space
  3. tables of vowel formant measurements
  4. probabilities for the preceding and following phonetic environments of each vowel token

There is also an option to manually check and correct the transcriptions or alignments.

How to cite

DARLA is designed and implemented by Sravana Reddy and James Stanford, with programming assistance from Irene Feng, and is supported by the Neukom Institute for Computational Science.

The completely automated system transcribes and aligns the input speech data using automatic speech recognition (ASR) techniques using the CMU Sphinx toolkit. We use the ProsodyLab Aligner for alignment, the FAVE-extract code for formant measurement, and the R vowels package for plotting.

Suggested in-text citation: DARLA (Reddy & Stanford 2015), using FAVE (Rosenfelder, Fruehwald, Evanini & Yuan 2011) and the Vowels R package (Kendall & Thomas 2010).

Suggested bibliographic entries:

  • Reddy, Sravana and James Stanford. 2015. Toward completely automated vowel extraction: Introducing DARLA. Linguistics Vanguard.
  • Rosenfelder, Ingrid; Fruehwald, Joe; Evanini, Keelan and Jiahong Yuan. 2011. FAVE (Forced Alignment and Vowel Extraction) Program Suite. fave.ling.upenn.edu.
  • Kendall, Tyler and Erik R. Thomas. 2010. Vowels: vowel manipulation, normalization, and plotting in R. R package. cran.r-project.org/web/packages/vowels/index.html.

In addition, if you use the NORM toolkit for custom normalization and plotting of the vowel space, please also cite:
